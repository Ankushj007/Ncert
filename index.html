APIs to Build This

Here are the best APIs and services you should use to implement this platform — from book storage to AI Q&A.

Official
NCERT Official Website API
NCERT publishes all books as free PDFs. You can directly link or scrape/download PDFs from their official site. No auth required for public books.
https://ncert.nic.in/textbook.php
AI Core
Anthropic Claude API
Best-in-class LLM for answering curriculum questions. Use Claude with NCERT PDF context to answer questions accurately with chapter citations.
https://api.anthropic.com/v1/messages
AI Alternative
Google Gemini API
Gemini Pro supports large context windows and can ingest full NCERT PDFs. Great for multi-subject, multi-chapter Q&A without chunking.
https://generativelanguage.googleapis.com/v1beta/
Free
Pinecone / Weaviate (Vector DB)
For RAG (Retrieval Augmented Generation): chunk NCERT PDFs, embed them, store in a vector DB. Query at runtime to fetch relevant passages before sending to AI.
https://api.pinecone.io | https://weaviate.io
Free Tier
OpenAI Embeddings API
Use text-embedding-3-small to create vector embeddings of NCERT content chunks. Combine with vector DB for semantic search over book content.
https://api.openai.com/v1/embeddings
Free
PDF.js / pdf-parse (PDF Parsing)
Client-side PDF rendering using Mozilla's PDF.js library. Parse and extract text from NCERT PDFs for embedding and search indexing.
https://cdnjs.cloudflare.com/ajax/libs/pdf.js/
Storage
AWS S3 / Cloudflare R2
Host mirrored NCERT PDFs for fast, reliable delivery. Cloudflare R2 has no egress fees — ideal for a high-traffic free education platform.
s3.amazonaws.com | r2.cloudflarestorage.com
Free
Firebase / Supabase (Auth + DB)
User authentication, bookmarks, reading history, and progress tracking. Both offer generous free tiers ideal for student platforms.
firebase.google.com | supabase.com
💡 Recommended Architecture
Step 1: Download all NCERT PDFs → host on Cloudflare R2
Step 2: Parse PDFs with pdf-parse → chunk into sections (500–1000 tokens)
Step 3: Embed chunks with OpenAI text-embedding-3-small → store in Pinecone
Step 4: On user question → embed query → vector search → top 5 relevant chunks
Step 5: Send chunks + question to Claude API → get accurate, cited answer
Step 6: Use Firebase for auth, bookmarks, and reading progress
