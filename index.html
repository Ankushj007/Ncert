APIs Powering This

The complete stack of APIs and services used in VidyaSetu.

Official
NCERT Official PDFs
All books served directly from NCERT's server. Free, authentic, always updated.
ncert.nic.in/textbook/pdf/[code].pdf
AI Core
Anthropic Claude API
Claude Haiku powers the AI tutor — fastest model with streaming support. Secured in Node.js backend.
api.anthropic.com/v1/messages
Free Tier
OpenAI Embeddings
text-embedding-3-small converts NCERT PDF chunks to vectors for semantic search (RAG).
api.openai.com/v1/embeddings
Free
Pinecone Vector DB
Store and query NCERT content embeddings. Fetch relevant passages before sending to Claude.
api.pinecone.io
Free
Firebase Auth + DB
User authentication, bookmarks, reading history, and progress tracking.
console.firebase.google.com
Storage
Cloudflare R2
Mirror NCERT PDFs for fast delivery with zero egress fees — ideal for a free student platform.
r2.cloudflarestorage.com
💡 Backend Architecture
All API keys live in .env on the Node.js backend — never exposed to the browser.
Frontend → /api/ask → server.js → Claude API (streamed back) → Browser
Frontend → /api/pdf → server.js → NCERT server → Direct download
